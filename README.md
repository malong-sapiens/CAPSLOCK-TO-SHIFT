# CAPSLOCK-TO-SHIFT

Permanent Caps Lock → Left Shift remapping using systemd-hwdb.

## Install

```bash
git clone https://github.com/malong-sapiens/CAPSLOCK-TO-SHIFT.git
cd CAPSLOCK-TO-SHIFT

chmod +x install.sh
sudo ./install.sh

sudo reboot
```

## Uninstall

```bash
cd CAPSLOCK-TO-SHIFT

chmod +x uninstall.sh
sudo ./uninstall.sh

sudo reboot
```
